# CALCULADORA

Projeto desenvolvido durante o curso de Flutter pela Raro Academy, com o objetivo de reproduzir uma calculadora com uma interface simples e operações básicas utilizando Flutter.

* Flutter
* Dart

## Conceitos Aplicados

* StatefullWidget
* MaterialApp
* setState
* GrigView
* gestureDetector
